# BASAHON 

how to run
1. go to file directory
2. run the cmd
3. command "composer install"
4. after that "npm install"
5. "npm run build"
6. then "php artisan key:generate"
7. run php artisan serve

make sure to:
*xampp is running properly.
*.env file has connection/access to the database.

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=beyou_brew_cafe
DB_USERNAME=root
DB_PASSWORD=
